Learn Google Cloud Platform (GCP)

In this document, I will explain what you need to know in order to work with Google Cloud Platform (GCP). It is the path, I have taken and it is not intended to be complete reference. However, I will try to include links to resources that I have used to learn and practice GCP.

Access to Google Cloud Platform

There are several ways to access and interact with Google Cloud Platform (GCP).

  • CLI: Command Line Interface software gcloud
  • Google Cloud Console: Access to GCP from graphical user interface to access from https://console.cloud.google.com
  • Google Cloud App: You can access to GCP from your phone by GCP app that can be downloaded from https://cloud.google.com/app
  • SDK: You can use SDKs for different programming languages such as Python to manage GCP
